# Consulting Design Language: Deloitte/McKinsey Style Rules

**Source**: Empathy Agent web research + analysis of consulting pitch book patterns
**Relevance**: Visual and structural standards for business-heavy investor documents

---

## The Consulting Page Structure

Every page follows this hierarchy:

```
[ACTION TITLE]    Complete sentence stating the page's conclusion (18-20px, bold)
[SUBHEADING]      Clarifies scope (optional, 12px)
[BODY]            Charts/tables that PROVE the action title
[SOURCE LINE]     Bottom of page, every data point attributed (7-8px)
[PAGE NUMBER]     Prominent, functional (9px monospace)
```

**Source**: McKinsey Pyramid Principle methodology; Deloitte client deliverable standards

---

## Grid System
- 12-column rigid grid (or 2/3-column modular)
- Every element snaps to grid -- no floating or atmospheric positioning
- Zero or minimal border-radius (2-4px maximum)
- All left edges share exact vertical alignment

**Source**: Empathy Agent analysis of Deloitte/BCG strategy deck templates

## Typography Rules

| Element | Consulting Standard |
|---------|-------------------|
| Headlines | Sans-serif 600/700, 16-20px, action titles (conclusions) |
| Body | Sans-serif 400, 10-11px, bullets and concise statements |
| Data | Monospace or tabular sans, right-aligned numbers |
| Section labels | 10-12px dark uppercase, functional wayfinding |
| Emphasis | Bold weight of same font + color highlight of key numbers |

**Key rule**: ONE font family for everything. Weight and size create hierarchy, not font switching.

**Source**: Empathy Agent typography analysis across Deloitte, McKinsey, BCG deck samples

## Color Usage

Maximum 3 functional colors:
- **Dark text** (near-black or dark brown) -- primary
- **One brand accent** -- used sparingly for emphasis
- **Gray** -- secondary information

Colors carry meaning, not decoration:
- Green/sage = positive, growth, target met
- Red/terra = negative, risk, gap
- Amber = highlight, the number to look at first
- Gray = context, secondary, supporting

**Source**: Deloitte brand guidelines analysis; McKinsey color restraint principles

## Chart Styles

Consulting firms use:
- Horizontal bar charts (easier label reading than vertical)
- Waterfall/bridge charts (financial flow composition)
- Simple line charts (time-series)
- Stacked bar charts (composition analysis)

Consulting firms do NOT use:
- Radial/donut charts
- Decorative data visualization
- Colored gradient fills
- Charts without source citations
- Charts without action title conclusions

Every chart has: title (states insight), labeled axes, annotation callouts, source line.

**Source**: Empathy Agent visual language research

---

## Table Standards

```
LEFT-ALIGNED labels | RIGHT-ALIGNED numbers
Consistent decimal alignment (tabular-nums)
Alternating row shading (3-5% opacity)
Subtotal rows: top border + bold weight
Total rows: top AND bottom border (double rule)
Assumption rows: lighter gray, indented, italic, "Assumption:" prefix
Units always labeled (EUR, %, months)
```

**Source**: Deloitte financial table formatting standards

---

## What Consulting Decks Do NOT Have

- Glassmorphic effects (backdrop-filter, blur)
- Atmospheric background gradients
- Pull quotes with decorative borders
- Editorial prose paragraphs
- Brand-forward language
- Decorative separators
- Rounded corners > 4px
- Multiple accent colors simultaneously
- Persona cards with narrative descriptions
- Dark inverted vision blocks
- Serif typography for headlines
- Large type sizes (40px+)
- "Mood" or "atmosphere" as design goals

**Source**: Empathy Agent anti-pattern analysis across consulting firm deliverables
