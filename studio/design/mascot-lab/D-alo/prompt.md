## VARIATIONS (expression-via-glow sheet)

### Idle (default chat state)
```
Same object Alo. Steady warm internal glow at standard intensity. Perfectly still. Slight ambient warmth radiating.
```

### Listening (user is typing)
```
Same object Alo. Internal glow slightly brighter (15% increase), pulse cycle faster. Capture mid-pulse expansion.
```

### Thinking (API pending)
```
Same object Alo. Internal glow shifting from warm amber toward soft violet-rose (capture mid-shift, a blended moment). Surface cloudiness more visible as if something is moving internally. Very slight rotation.
```

### Aligned (user submits)
```
Same object Alo. Internal glow at brightest — honey-gold, nearly 40% opacity outward halo. The orb raised 8px higher than idle (floating slightly more). Capture the single moment of maximum brightness.
```

### Resting (end of session)
```
Same object Alo. Internal glow dimmed to 30% of standard intensity. Surface slightly darker, cooler. Shadow softer. The whole object feels asleep but not off.
```
